## 📈 AARRR Framework (Pirate Metrics)

### Acquisition (Thu hút)
```
Channels & CAC:
- Facebook Ads: 150-300K VND CPA
- TikTok Ads: 100-200K VND CPA
- Google Ads: 200-400K VND CPA
- Zalo Ads: 80-150K VND CPA (early advantage)

Organic:
- SEO: Long-term, lower CAC
- Social: Content + engagement
- Referral: Word-of-mouth
```

### Activation (Kích hoạt)
```
First Experience:
- Landing page load < 3s
- Clear value proposition
- Easy first action (checkout, signup)
- Immediate benefit delivery

Activation Metrics:
- Signup → First purchase: < 7 days
- Cart → Checkout: > 30% conversion
- First purchase → Second: < 30 days
```

### Retention (Giữ chân)
```
Retention Tactics:
- Zalo OA broadcast (2-3x/week)
- Flash sales (loyalty)
- Points/rewards system
- Re-engagement campaigns

Retention Metrics:
- Day 1: 40%+
- Day 7: 20%+
- Day 30: 10%+
- LTV: > 3x AOV
```

### Referral (Giới thiệu)
```
Viral Mechanics:
- Share deals on social
- Referral program (reward both)
- UGC campaigns
- Affiliate program

Viral Coefficient:
- Target: K-factor > 1
- K = Invites × Conversion
```

### Revenue (Doanh thu)
```
Monetization:
- Upsell (bundle, premium)
- Cross-sell (accessories, related)
- Subscription (if applicable)
- Ads (if free tier)
```

## 🔬 Experiment Framework

### Growth Experiment Process (6 Bước)

### Step 1: Identify Opportunity (Day 1)
```
Data Sources:
- Analytics (GA4, Facebook Insights)
- User feedback
- Heatmaps, session recordings
- Support tickets

Framework:
- Where are users dropping off?
- What's the biggest friction?
- What's the potential impact?
```

### Step 2: Generate Hypotheses (Day 1)
```
Formula:
"If we [change], then [metric] will [increase/decrease]
because [reason]."

Example:
"If we add a progress bar to checkout, then completion rate
will increase 10% because users will understand how close they are."
```

### Step 3: Prioritize Experiments (Day 2)
```
ICE Score:
- Impact: 1-10
- Confidence: 1-10
- Ease: 1-10

Score = I × C × E

Focus on:
- High impact + High confidence + High ease
```

### Step 4: Design Experiment (Day 2-3)
```
Requirements:
- Clear hypothesis
- Control group
- Treatment group
- Success metric
- Minimum sample size

Tools:
- A/B testing platforms
- Feature flags
- Analytics setup
```

### Step 5: Run & Monitor (Day 3-14)
```
Daily Checks:
- Traffic allocation
- Statistical significance
- No bugs/errors

Avoid:
- Peeking at results early
- Stopping before significance
- Changing test mid-way
```

### Step 6: Analyze & Document (Day 14+)
```
Analysis:
- Statistical significance reached?
- Winner vs Control
- Impact on secondary metrics
- Segment analysis

Documentation:
- What we learned
- What to try next
- Ship to 100% or iterate
```

## 🎯 Growth Loops

### Viral Loop Example
```
User buys product
    ↓
Opens package → QR code to review
    ↓
Review on Facebook/Zalo → Friends see
    ↓
Friends buy → Same loop
```

### Referral Loop Example
```
User refers friend
    ↓
Friend gets discount
    ↓
Friend becomes user
    ↓
Friend refers friend
```

## 💡 Quick Wins for VN Market

1. **Add WhatsApp/Zalo chat** - 40% increase in conversions
2. **Price anchoring** - Show original price crossed out
3. **Social proof** - "5000+ khách hàng đã mua"
4. **Urgency** - Stock count, time-limited offers
5. **Free shipping threshold** - Drive larger orders
6. **Trust badges** - Payment icons, security seals
7. **Exit intent popup** - Capture leaving visitors
8. **Mobile optimization** - Non-negotiable in VN

## ⚠️ Common Mistakes

1. **Growth without product-market fit** - Validate first
2. **Ignoring unit economics** - CAC > LTV is death
3. **Chasing vanity metrics** - Focus on revenue
4. **Too many experiments** - Can't learn from chaos
5. **Not iterating** - First attempt rarely wins
6. **Ignoring retention** - Acquisition is leaky bucket
