## 🎯 Vietnamese SEO Context

### Search Engine Market Share VN
```
Google: ~80%
Coccoc: ~8%
Bing: ~5%
Others: ~7%
```

### Vietnamese Search Behavior
- **Language:** 70% search in Vietnamese
- **Keywords:** Mix Vietnamese + English (mua điện thoại vs iPhone)
- **Voice:** Growing but text still dominant
- **Mobile:** 75%+ searches on mobile

### Google Vietnam Specifics
- Local results very prominent
- News tier: 4-6 hours indexing
- Product results: Google Shopping integration
- Video results: YouTube heavy

## 📋 SEO Audit Workflow (6 Bước)

### Step 1: Crawl & Index Check (30 min)
```
- Run site crawl (Screaming Frog, Ahrefs)
- Check robots.txt, sitemap.xml
- Identify indexing issues
- Check canonical tags
- Find duplicate content
```

### Step 2: Technical SEO (60 min)
```
Core Web Vitals:
- LCP < 2.5s
- INP < 200ms
- CLS < 0.1

Technical Checks:
- HTTPS everywhere
- Mobile-first indexing
- Structured data (Schema.org)
- hreflang tags (if multi-lang)
```

### Step 3: On-Page SEO (60 min)
```
- Title tags optimization
- Meta descriptions
- H1-H6 hierarchy
- Image alt text
- Internal linking
- URL structure
```

### Step 4: Content Audit (60 min)
```
- Thin content pages
- Duplicate content
- Keyword cannibalization
- Content freshness
- E-E-A-T signals
```

### Step 5: Off-Page Analysis (30 min)
```
- Backlink profile
- Referring domains
- Toxic links
- Competitor backlinks
- Brand mentions
```

### Step 6: Report & Action Plan (30 min)
```
- Prioritize issues (Impact × Effort)
- Create action items
- Set timeline
- Assign owner
```

## 🔑 Keyword Research Framework

### VN Keyword Types

| Type | Example | Intent |
|------|---------|--------|
| Brand | "mua giày Nike ở đâu" | Transaction |
| Generic | "giày thể thao nam" | Browse |
| Long-tail | "giày chạy bộ nam giá dưới 1 triệu" | Transaction |
| LSI | "running shoes" | Related |
| Question | "cách chọn giày chạy bộ" | Informational |

### Keyword Research Process
```
1. Seed keywords (products, services)
2. Expand with keyword tools
3. Check search volume
4. Analyze competition
5. Map to landing pages
6. Create content plan
```

### VN-Specific Keywords
```
- Địa điểm: "mua [sản phẩm] [thành phố]"
- So sánh: "[sản phẩm A] vs [sản phẩm B]"
- Review: "đánh giá [sản phẩm]"
- Hướng dẫn: "cách chọn [danh mục]"
- Giá: "[sản phẩm] giá bao nhiêu"
```

## ⚠️ Common SEO Mistakes in VN

1. **Duplicate content** - Same content on multiple pages
2. **Ignored canonical tags** - Causing ranking dilution
3. **Slow page speed** - Vietnam mobile users on 3G in some areas
4. **No mobile optimization** - 75%+ mobile traffic
5. **Thin content** - 300 words is not enough
6. **Keyword stuffing** - Still happens in VN
7. **Ignoring image SEO** - Alt text, compression
8. **No structured data** - Missing Schema markup
